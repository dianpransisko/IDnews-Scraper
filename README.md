📰 ID News Scraper - Privat
Monitoring Berita Otomatis via Supabase & React Dashboard

Proyek ini adalah sistem pemantauan berita otomatis yang dirancang untuk mengumpulkan data berita terkini (fokus pada Pemilu 2029) dari berbagai media digital Indonesia, menyimpannya ke database cloud, dan menampilkannya di dashboard pribadi yang terlindungi password.

🚀 Fitur Utama
Automated Scraper: Script Python yang berjalan otomatis menggunakan GitHub Actions (Cron Job).
Cloud Database: Integrasi dengan Supabase untuk penyimpanan data berita secara real-time.
Protected Dashboard: Frontend berbasis React yang dideploy di Vercel dengan akses terbatas.
Responsive UI: Tampilan kartu berita yang nyaman dibaca di perangkat mobile maupun iPad.
